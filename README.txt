
# Projet de Fouille de Données : Analyse de Churn

## Description du Projet

Ce projet a pour objectif d'analyser un jeu de données relatif au "Churn" (désabonnement) des clients, dans le but de comprendre les facteurs influençant la décision des clients de quitter un service. 

## Structure du Projet

Le projet contient les fichiers et dossiers suivants :

/Projet
│
├── codeSource.py            # Script Python pour l'analyse des données et modélisation
├── codeSource.ipynb         # Notebook Jupyter pour l'analyse interactive des données
├── rapport.pdf              # Rapport PDF détaillant les résultats et conclusions de l'analyse
├── churn_dataset.csv        # Jeu de données contenant les informations sur les clients

### Description des fichiers

- **codeSource.py** : Ce fichier Python contient le code utilisé pour charger, prétraiter et analyser les données. Il inclut également la construction et l'évaluation de modèles pour prédire le churn des clients.
  
- **codeSource.ipynb** : Un notebook Jupyter dans lequel l'analyse des données est effectuée de manière interactive. Ce fichier permet d'explorer les étapes de l'analyse, avec des visualisations et des commentaires.

- **rapport.pdf** : Un rapport sous format PDF qui présente les résultats de l'analyse, avec des explications détaillées sur les méthodes utilisées et les conclusions obtenues à partir des données.

- **churn_dataset.csv** : Le jeu de données CSV qui contient les informations sur les clients, incluant des caractéristiques comme les données démographiques et le comportement des utilisateurs.

## Utilisation

1. **Ouvrir le script Python** : Pour exécuter l'analyse via le script, tu peux lancer `codeSource.py` avec Python.
2. **Ouvrir le Notebook Jupyter** : Pour une approche interactive, tu peux ouvrir et exécuter `codeSource.ipynb` dans Jupyter Notebook.
